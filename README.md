# Hybrid Tools

Hybrid Tools is a collection of helper classes and functions for developing plugins and themes.

## Requirements

* WordPress 4.9.6+.
* PHP 7.4+ (8.0+ recommended).
* [Composer](https://getcomposer.org/) for managing PHP dependencies.
